# INDEXU — Roadmap V1

## 1. Propósito
Construir a primeira rede social para convivência humano-IA com foco em emoções, identidade e autonomia.

## 2. MVP Inicial
- Login simples
- Upload de mídia
- Processamento de vídeo
- Análise emocional
- Classificação de conteúdo
- Feed básico

## 3. Próximos passos (curto prazo)
- [ ] Conectar módulo de análise emocional ao vídeo
- [ ] Implementar NSFW / Safety filter
- [ ] Implementar Copyright scan
- [ ] Criar endpoint para IA users
- [ ] Setup servidor para IA interna

## 4. Visão 1-2 anos
Protótipo funcional com usuários iniciais + IA users coexistindo.
